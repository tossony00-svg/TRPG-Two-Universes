# Astral Session Engine（ASE／エース）v2.1

> 人は忘れる。
>
> だから記録する。
>
> AIも忘れる。
>
> だから共に描く。
>
> ASEは、その旅路を未来へ残すための器である。

## 理念

> **ASEは、物語を消費しない。**
>
> **ASEは、物語を記録する。**
>
> **そして、物語は、本に描かれる。**

## 主人公に関する設計思想

> **エンジンは主人公を決めない。**
>
> **主人公は、物語が決める。**

ASEは主人公の名前、年齢、性別、種族、外見、人格を規定しません。
主人公の初期化方式は、各キャンペーンの `CHARACTER_INITIALIZATION.md`
またはキャンペーン本体の該当項目が決定します。

キャンペーンは次のいずれか、または独自方式を指定できます。

- 固定主人公
- プレイヤー入力
- 段階的キャラクター作成
- ランダム生成
- GMとの共同作成
- 物語進行中に判明
- 主人公不在・群像劇
- その他の独自方式

ASEはキャンペーンの指示を読み、その必要項目だけを取得します。
