# 二つの宇宙(そら)の狭間で

**ASE(Astral Session Engine) v2.1 対応キャンペーンパッケージ**

---

## これは何か

AIをゲームマスター(GM)として、剣と魔法の異世界を旅するTRPGキャンペーンです。

D&D準拠のd20判定システムを土台にしつつ、「転生」「宇宙の均衡」「大切な人との絆」を軸にした物語が展開します。

このリポジトリには、AI GMが物語を進行・記録・復元するために必要な仕様一式(ASEエンジン)と、このキャンペーン固有の世界観設定が収められています。

---

## フォルダ構成

| フォルダ / ファイル | 内容 |
|---|---|
| `ASE_ENGINE/` | AI GMの運用ルール(場面転換の型・記録方式・復旧手順など) |
| `CAMPAIGN/` | このキャンペーン固有の世界観・進行フラグ・冒険記録テンプレート |
| `SAVES/` | セッション中断時のSaveデータ |
| `STORY_LOG/` | 確定した本文・会話を原文保存する記録 |
| `ASE_MASTER_新規プレイ_最新版.md` | 上記すべてを結合した、AIに渡す用の統合ビルドファイル |
| `ASE_Players_Guide_v2.2_旅の手引き.md` | プレイヤー向けの使い方ガイド |
| `README_FIRST.md` | 新規プレイ開始時の簡単な案内 |

---

## 遊び方

### 新しく始める

1. ASE対応のAI(Claudeなど)との新しい会話を開始する
2. `ASE_MASTER_新規プレイ_最新版.md`(またはこのリポジトリ一式)を添付する
3. 次の文章を送信する

   ```
   ASE展開。

   キャンペーン仕様に従って新規セッション開始。
   ```

### 続きから再開する

1. 新しい会話を開始する
2. `ASE_MASTER_新規プレイ_最新版.md`と、最新のSave(または冒険記録・進行フラグ)を添付する
3. 次の文章を送信する

   ```
   ASE展開。

   最新セーブを読み込み、
   セッション再開。
   ```

詳しい使い方は `ASE_Players_Guide_v2.2_旅の手引き.md` を参照してください。

---

## 注意

- 進行フラグ・冒険記録・Saveは、プレイが進むたびに更新・追加されるファイルです。プレイ後は、最新の状態をこのリポジトリに反映しておくことをおすすめします
- ネタバレを含む設定(GM専用セクション)が含まれるため、プレイヤーとして遊ぶ場合は `CAMPAIGN/WORLD.md` の内容を事前に読みすぎないようご注意ください
